🎯 QUICK START (5 MENIT):
════════════════════════════════════════════════════════════════════════════

1. EXTRACT PROJECT
   $ unzip nimestream-v2.0.zip
   $ cd nimestream

2. INSTALL DEPENDENCIES
   $ npm install

3. SETUP FIREBASE (Optional, tapi recommended)
   - Buat Firebase project di https://console.firebase.google.com
   - Enable Authentication, Firestore, Storage
   - Copy credentials ke public/js/config.js

4. RUN SERVER
   $ npm start
   
   Server akan berjalan di: http://localhost:3000

5. BUKA DI BROWSER
   Pergi ke: http://localhost:3000

🆘 TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════════════

"npm: command not found"
  → Install Node.js dari https://nodejs.org

"Port 3000 already in use"
  → Edit server.js, ubah PORT ke 3001 atau port lain

"Cannot find module"
  → Jalankan: npm install

"Firebase connection error"
  → Pastikan credentials di config.js benar
  → Atau jalankan tanpa Firebase (aplikasi tetap berfungsi)

"Blank page"
  → F12 → Console → cek error message
  → Refresh browser dengan Ctrl+Shift+R
